# CarRegNumTest

# Prerequisites:

1. Maven
2. Java
3. IntelliJ IDE

## Run the project:

- In 'CarRegNumPage.java' -> TestResults method is annotated with @Test
- Right click on that method and then click on the option 'Run TestResults()'
   (or)
   From the IntelliJ IDE top menu, click on the 'Run' button
